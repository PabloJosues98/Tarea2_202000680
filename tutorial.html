Creacion de clase y como instanciarlo

Pasos para crear una clase en java

  • Dentro del proyecto podemos buscar el paquete en el cual esta nuestra clase principal, este tiene forma de una caja de carton como se ve en la imgaen a continuacion
  • Al hacer clic derecho sobre el paquete se despliegan mas opciones, ingresamos en NEW y presionamos en JAVA CLASS
  • Se despliega un recuadro en el cual debemos colocar el nombre de la clase este debe ser un nombre que represente al objeto en la vida real
  • Luego presionamos en finalizar
  • Se nos creara una clase vacia como se muestra a continuacion
  • Ahora debemos crear los atributos en esta clase, estos seran valores que queremos que guarden o queremos obtener de este
  • A continuacion se crearan los constructores, uno vacio en el que se asignaran valores por defecto y uno que recibe parametros
  • Tambien creamos un metodo el cual nos dara la nota que suena en una guitarra dependiendo la cuerda y el traste que se presione este metodo hace mension a otro metodo llamado selecionNota
  • Este es el metodo selecionNota este metodo genera imprime en consola la nota

Pasos para Instanciar una clase

  • En nuestra clase donde queremos tener acceso de la clase que acabamos de crear, se debe realizar de la siguiente manera, se crea un objeto del tipo de clase(es decir se coloca tal y como se llama la clase que deseamos instanciar) y le colocamos un nombre con el cual lo nombraremos detro de esta clase, luego colocamos "=" y "new" el cual permite la instancia de la clase, luego va de nuevo el nombre de la clase, en este ejemplo se le mando parametros para que se inicialize con esos valores y no con los de fabrica
  • Ahora que ya esta instanciado, la manera de poder acceder a su metodo llamado nota es: colocar el nombre con el que lo llamamos en esta clase, es decir el nombre que colocamos antes del igual al momento de instanciarlo, seguido del punto, el cual nos da acceso a los metodos de la clase, el metodo nota recibe como parametro traste y cuerda por ello se deben de proporcionar al metodo como se muestra a continuacion
  • Ahora solo queda probar nuestras instancias ejecutando el programa
